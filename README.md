# ✈ Airline Management System

## 📌 Overview
A Java Swing-based desktop application for managing airline bookings, flights, and users.

## 🚀 Features
- Admin & User Login
- User Registration
- Flight Management
- Flight Search & Booking
- Booking Cancellation
- Booking History
- Passport Validation

## 🛠️ Technologies
- Java
- Java Swing
- AWT
- Collections
- Regex

## ▶️ Run
Compile and run `AirlineManagementSystem.java`.
